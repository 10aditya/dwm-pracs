# dwm-pracs

## All DWM practicals

### Command to install python3 on Ubuntu

```
$ sudo apt-get install python3
```

<br>

### Command to install pip3 on Ubuntu

```
$ sudo apt-get -y install python3-pip
```

<br>

### If you get ModuleNotFoundError:

```
$ pip3 install <ModuleName>
```

<br>

### To run python file:

```
$ python3 filename.py
```

<br>
